観察事実
Advisor High「Security」カテゴリで Microsoft.Compute/virtualMachines に 37 件の推奨事項が存在する。
影響
認証・暗号化・ネットワーク設定の不備が複合した場合、攻撃成功率が高まる可能性がある。
推奨対応
Azure Advisor の Security セクションから対象推奨事項を確認し、重複する VM があればリスク集中として優先対応する。Defender の High 推奨との照合を行うこと。
高 優先度 3 ServicePrincipal の広域スコープ割り当て
観察事実
ロール割り当て 41 件のうち 37 件が ServicePrincipal に対するもので、Subscription スコープ 28 件・ManagementGroup スコープ 13 件を含む。
影響
過剰な特権を持つ SP のクレデンシャルが漏洩した場合、環境全体への横断的アクセスが可能になるリスクがある。
推奨対応
各 SP の用途・発行者・有効期限を棚卸しし、最小権限の原則に基づいてスコープをリソースグループ以下へ縮小できるか検討する。不要な SP は速やかに無効化する。

観察事実

影響
運用品質に関する設定不備は障害時の復旧遅延やサービス停止につながる可能性がある。
推奨対応
Advisor の OperationalExcellence 推奨事項を確認し、監視・診断設定・バックアップポリシーの有無を点検する。
中 優先度 5 ディスク可用性推奨（Advisor High / HighAvailability）

推奨対応
Advisor の HighAvailability セクションから対象ディスクを確認し、冗長化（ZRS・スナップショット等）の適用を検討する。本番ワークロードに紐づくディスクを優先すること。
