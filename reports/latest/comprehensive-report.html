エグゼクティブサマリ

本レポートは、現在ログイン中の Azure サブスクリプションにおけるリソース、RBAC、NSG、Defender for Cloud、Azure Advisor の横断的な棚卸しとリスク分析を実施した結果をまとめたものです。

主要数値

  • リソース総数: 75 件 / リソースグループ数: 9 / リージョン数: 5
  • VM 件数: データなし / Storage (LRS) 件数: データなし / Public IP 件数: データなし
  • タグ付与率: 52.0%
  • RBAC 総割り当て: データなし / Owner 割り当て数: 13 / UAA 割り当て数: 2 / 孤児候補件数: 40 / MG スコープの Owner: データなし
  • NSG 数: データなし / 総ルール数: データなし / 高リスクルール件数: 0
  • Defender for Cloud Unhealthy 件数: 155（High: 39、Medium: 未集計、Low: 未集計）
  • Defender 最頻出推奨事項: Update libncurses6, Update ncurses-term, Update libncursesw6 など複数の VM セキュリティ更新
  • Advisor 件数: データなし（5カテゴリ内訳: データなし）、High Impact 件数: 29
  • Advisor 最頻出問題: EDR solution should be installed on Virtual Machines
  • Defender for Cloud および Azure Advisor は有効かつデータ取得可能です。

主要改善領域

  1. 公開境界：NSG 高リスクルールは 0 件ですが、Public IP 件数は未取得のため、公開境界の露出リスクは要監視です。
  2. 特権境界：Owner 割り当て数 13 件、孤児候補 40 件と特権ロールの恒久付与と blast radius に注意が必要です。
  3. 監視フィードバック：Defender for Cloud で High レベルの不健康状態が 39 件、Advisor で High Impact が 29 件存在し、監視とフィードバックループの強化が求められます。
  4. ガバナンス基盤：タグ付与率 52.0% と半数程度であり、タグ必須化やポリシー適用によるガバナンス強化が必要です。

2. 特権ロールの恒久付与と blast radius

推奨: Foundational CSPM の導入、Secure Score の月次レビューを実施し、継続的な改善を図ること。

  • Azure Defender for Cloud の概要
  • Secure Score の活用
